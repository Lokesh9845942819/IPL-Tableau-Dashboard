#  IPL Data Analysis Dashboard (Tableau)

##  Project Overview

This project analyzes Indian Premier League (IPL) match data using Tableau. The dashboard provides insights into team performance, player statistics, match results, and tournament trends through interactive visualizations.

---

##  Objectives

- Analyze team performance.
- Identify top-performing batsmen and bowlers.
- Compare team wins.
- Understand venue performance.
- Create an interactive dashboard.

---

##  Tools Used

- Tableau
- Microsoft Excel / CSV

---

##  Dashboard Features

- Team Performance
- Top Run Scorers
- Top Wicket Takers
- Match Winners
- Toss Analysis
- Venue Analysis
- Season-wise Statistics
